A-API??

OwO Ipsum provides an open-source and free API. You even don't need API key!

You can use it for any purpose:

  • Software development
  • Design
  • Having fun
  • Wherever it would be useful

Documentation

See requests and responses examples on GitHub (with syntax highlighting).


OwO Ipsum API provides following endpoints.

1. GET /words

Generates and returns words.

Query

Parameter Effect Type Default Possible values
number Number of generated words integer 0 [0, 100]
format Response format string "json" "json", "html", "plain"

2. GET /sentences

Generates and returns sentences.

Query

Parameter Effect Type Default Possible values
number Number of generated words integer 0 [0, 100]
words Sentence length - how many words in it integer or range Random (5, 10) [1, 15] or range
stutter Probability of stutter appearance float 0.1 [0, 1]
face Probability of face appearance float 0.05 [0, 1]
action Probability of action appearance float 0.05 [0, 1]
format Response format string "json" "json", "html", "plain"

Range

Range - string containing two integers separated by dash. Examples: 3-6, 9-15. In general terms: <start>-<end> Means that random number from half-closed interval [<start>, <end>) will be taken.
